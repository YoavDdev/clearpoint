# איך ליצור חשבונית עם מנוי חודשי

## צעדים:

### 1. לך לדף הלקוח
```
/admin/customers/[id]
```

### 2. צור חשבונית חדשה עם InvoiceCreator
- הוסף פריטים (ציוד, התקנה)
- **סמן את התיבה: "כולל מנוי חודשי" ✅**
- הגדר מחיר חודשי (למשל: ₪149)

### 3. מה קורה:
```javascript
includeSubscription: true  ✅
monthlyPrice: 149

→ יוצר:
  1. חשבונית (ציוד + התקנה)
  2. מנוי חודשי ב-DB
  3. תשלום חוזר ב-PayPlus
```

### 4. המנוי נוצר אוטומטית כש:
- התשלום הראשון (של הציוד) מצליח
- הקוד יוצר רשומה ב-`subscriptions` table:
  ```sql
  INSERT INTO subscriptions (
    user_id,
    status = 'active',
    amount = 149,
    billing_cycle = 'monthly',
    next_billing_date = NOW() + INTERVAL '1 month'
  )
  ```

### 5. תוצאה:
✅ למשתמש יש מנוי פעיל
✅ גישה למצלמות מופעלת
✅ חיוב אוטומטי כל חודש

---

## חשוב!
**"חודש ראשון חינם"** = לא מחייבים את המנוי החודשי בתשלום הראשוני.
התשלום הראשון = רק ציוד + התקנה.
החיוב החודשי הראשון = בעוד חודש.
