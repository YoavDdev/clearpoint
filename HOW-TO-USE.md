## 🚀 דרך 1: תשלום מושלם (מומלץ!)

### **התקנה + מנוי חודשי במקום אחד**

```
1️⃣ אדמין → /admin/customers/[id]
   ↓
2️⃣ גלול ל: "💎 תשלום מושלם"
   ↓
3️⃣ הוסף פריטי התקנה:
   • 4 מצלמות: ₪1,400
   • NVR: ₪800
   • עבודה: ₪500
   ↓
4️⃣ לחץ: "צור תשלום התקנה + מנוי"
   ↓
5️⃣ העתק לינק ושלח ללקוח
   ↓
6️⃣ לקוח משלם ₪2,700 (התקנה)
   + הכרטיס נשמר לחיובים חודשיים
   ↓
7️⃣ מנוי מופעל אוטומטית!
   ↓
8️⃣ כל חודש: Grow מחייב ₪149 אוטומטית
```

### **למה זה מושלם?**
✅ פעולה אחת במקום שתיים  
✅ לקוח משלם פעם אחת  
✅ הכרטיס נשמר אוטומטית  
✅ מנוי מופעל אוטומטית  
✅ חיובים חודשיים אוטומטיים

## 📋 דרך 2: נפרד (אופציונלי)

### **A. רק חשבונית התקנה**
```
1. כנס ל: /admin/customers/[id]
2. גלול ל: "InvoiceCreator"
3. הוסף פריטים
4. צור חשבונית
5. שלח ללקוח
```

### **B. רק מנוי חודשי**
```
1. כנס ל: /admin/customers/[id]
2. גלול ל: "SubscriptionManager"
3. לחץ "הפעל חיוב אוטומטי"
4. לקוח משלם פעם ראשונה
5. מחויב אוטומטית כל חודש
```
